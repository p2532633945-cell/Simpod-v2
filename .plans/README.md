# 项目计划文件夹

本文件夹存放 Simpod v2 的所有执行计划。

## 文件说明

| 文件 | 说明 | 状态 |
|------|------|------|
| `phase_3_核心功能完善.plan.md` | Phase 3 执行计划（热区隔离、复盘界面、用户系统） | ⏳ 进行中 |

## 如何使用

### 查看计划
```bash
cat .plans/phase_3_核心功能完善.plan.md
```

### 执行计划
在 Cursor 中使用命令：
```bash
/execute .plans/phase_3_核心功能完善.plan.md
```

## 计划生命周期

1. **创建** — 在此文件夹中创建新计划
2. **执行** — 使用 `/execute` 命令执行
3. **完成** — 移至 `PROGRESS.md` 的"已完成的工作"部分
4. **归档** — 可选：创建 `archived/` 子文件夹存放历史计划
