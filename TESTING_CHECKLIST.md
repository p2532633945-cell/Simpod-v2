### P5-4：热区编辑功能

**测试入口**：登录后进入任意播放页面 `/workspace/[id]`，查看右侧热区列表

- [ ] **编辑按钮显示**
  - 鼠标悬停在热区卡片上，右上角出现编辑图标（铅笔图标）
  - 图标默认隐藏，hover 才显示

- [ ] **打开编辑模态框**
  - 点击编辑图标，弹出编辑模态框
  - 模态框显示当前热区的 Start Time、End Time、Title、Description
  - 点击背景蒙层或 Cancel 按钮能关闭

- [ ] **编辑时间范围**
  - 修改 Start Time（秒），下方实时显示格式化时间（如 `01:23`）
  - 修改 End Time，同理
  - 输入 Start ≥ End 时出现红色错误提示，Save 按钮变灰
  - 时间合法时可正常保存

- [ ] **编辑标题和描述**
  - 在 Title 输入框输入文字
  - 在 Description 文本框输入多行文字
  - 点击 Save，模态框关闭，列表刷新显示修改后内容

- [ ] **删除热区**
  - 点击模态框左下角红色垃圾桶图标
  - 出现确认提示："Are you sure..."
  - 点击 Cancel 取消删除
  - 点击 Delete，热区从列表和波形图消失

- [ ] **用户隔离**
  - 用户 A 无法编辑用户 B 的热区（API 返回 403）

### P5-5：热区回溯功能

**测试入口**：进入任意播放页面 `/workspace/[id]`，查看播放器控制区

- [ ] **快退/快进按钮**
  - 播放控制区显示 `◁5s`、`◁10s`、`◁30s` 按钮（播放键左侧）
  - 显示 `5s▷`、`10s▷`、`30s▷` 按钮（播放键右侧）
  - 点击 `◁5s`：进度条向前跳 5 秒，时间显示更新
  - 点击 `10s▷`：进度条向后跳 10 秒，时间显示更新
  - 点击 `◁30s`：进度条向前跳 30 秒（不超过 0 秒）
  - 点击 `30s▷`：进度条向后跳 30 秒（不超过总时长）

- [ ] **进度条拖拽**
  - 鼠标点击进度条任意位置，音频跳转到对应时间
  - 按住鼠标拖拽进度条，时间实时更新

- [ ] **键盘快捷键**
  - 在非输入框状态下，按 `←`：音频后退 10 秒
  - 按 `→`：音频快进 10 秒
  - 按 `Space`：播放/暂停切换
  - 在输入框（如搜索框）内按方向键：不触发音频跳转

- [ ] **按钮 tooltip**
  - 鼠标悬停在按钮上，显示提示文字（如 "← 后退 10 秒"）

### P5-7：转录编辑功能

**测试入口**：登录后进入 `/hotzones` 复盘页面，找到有转录文本的热区卡片

- [ ] **转录显示**
  - 热区卡片显示 "TRANSCRIPT" 标签
  - 转录文本以只读灰色背景显示

- [ ] **进入编辑模式**
  - 点击铅笔图标（Edit），文本区域变为可编辑的 textarea
  - textarea 自动获得焦点
  - 出现撤销（↺）、重做（↻）、取消（×）、保存（✓）按钮

- [ ] **撤销/重做**
  - 修改文本后点击 ↺，恢复上一步
  - 点击 ↻，重做
  - 撤销到初始状态时 ↺ 变灰

- [ ] **保存**
  - 有修改时保存按钮为高亮绿色，无修改时变灰
  - 点击 ✓，显示 "Saved" 绿色提示，退出编辑模式
  - 刷新页面后，修改后的转录文本仍然显示

- [ ] **取消**
  - 修改文本后点击 ×，恢复原始文本，退出编辑模式

- [ ] **注释（Note）**
  - 点击气泡图标（MessageSquare），展开注释输入框
  - 在编辑模式下可输入注释
  - 保存后刷新，注释仍然存在
  - 再次点击气泡图标，收起注释区域

- [ ] **用户隔离**
  - 用户 A 无法编辑用户 B 的转录（API 返回错误）

### P5-8：转录搜索

**测试入口**：登录后进入 `/hotzones` 复盘页面，查看 header 右侧搜索框

- [ ] **搜索框交互**
  - 点击搜索框出现下拉面板
  - 输入关键词，实时显示匹配结果
  - 点击搜索框外部，下拉面板关闭
  - 输入内容后出现 × 清除按钮，点击清空

- [ ] **搜索结果**
  - 结果面板顶部显示 "X results"
  - 每条结果显示：时间范围、状态标签、转录上下文
  - 关键词在上下文中被**高亮**显示（绿色背景）
  - 无匹配时显示 "No matches found for \"xxx\""
  - 点击结果，跳转到对应播放器页面并自动播放

- [ ] **搜索历史**
  - 执行一次搜索（按 Enter 或点击结果）
  - 清空搜索框，点击搜索框，显示 "Recent Searches" 历史列表
  - 最多保存 10 条历史
  - 点击历史条目，自动填充并搜索
  - 点击 "Clear" 清除所有历史
  - 刷新页面后，搜索历史仍然存在（localStorage）

- [ ] **搜索范围**
  - 搜索转录文本内容能找到匹配热区
  - 搜索 audio_id 片段也能匹配

### P6-3：智能热区时间范围档位

- [ ] **旧热区不受影响**
