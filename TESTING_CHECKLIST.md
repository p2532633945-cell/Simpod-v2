# Simpod v2 当前版本测试清单

> **当前版本状态**：Phase 4 完成，Phase 5 待开始
> **测试目标**：验证核心功能稳定性，识别风险，为后续开发奠定基础

---

## 📋 测试范围

### ✅ 已完成功能（需验证）

#### 1. 用户系统
- [ ] **登录功能**
  - [ ] 正确的邮箱和密码能成功登录
  - [ ] 错误的邮箱或密码显示错误提示
  - [ ] 登录后自动跳转到首页
  - [ ] 登录状态在刷新后保持

- [ ] **注册功能**
  - [ ] 新邮箱能成功注册
  - [ ] 已存在的邮箱显示错误提示
  - [ ] 密码少于 6 个字符显示错误提示
  - [ ] 注册后收到验证邮件

- [ ] **密码重置**
  - [ ] 点击"Forgot password?"进入重置页面
  - [ ] 输入邮箱后收到重置链接
  - [ ] 点击重置链接能重置密码

- [ ] **个人资料**
  - [ ] 能访问 `/profile` 页面
  - [ ] 能编辑显示名称
  - [ ] 能查看账户状态（邮箱、验证状态、创建时间）
  - [ ] 能安全登出

#### 2. 播客搜索和播放
- [ ] **搜索功能**
  - [ ] 搜索"BBC"返回结果
  - [ ] 搜索同一个播客第二次立即返回（缓存命中）
  - [ ] 搜索结果显示播客名称、作者、封面

- [ ] **播客详情**
  - [ ] 点击搜索结果进入播客详情页
  - [ ] 显示播客信息和剧集列表
  - [ ] 剧集列表显示标题、发布日期、时长

- [ ] **音频播放**
  - [ ] 点击剧集进入播放器
  - [ ] 音频能正常加载和播放
  - [ ] 播放按钮能暂停和继续播放
  - [ ] 进度条能拖拽跳转
  - [ ] 变速按钮能改变播放速度（1x, 1.25x, 1.5x, 2x）

#### 3. 热区功能
- [ ] **创建热区**
  - [ ] 点击 MARK 按钮创建热区
  - [ ] 热区显示在波形图上
  - [ ] 热区显示在右侧热区列表
  - [ ] 热区包含转录文本

- [ ] **热区隔离**
  - [ ] 用户 A 创建的热区在用户 B 账户中不可见
  - [ ] 切换不同剧集，热区标记位置改变
  - [ ] 同一剧集的热区位置一致

- [ ] **热区播放**
  - [ ] 点击热区能跳转到该时间点
  - [ ] 从复盘页面点击热区能自动播放

- [ ] **复盘页面**
  - [ ] 能访问 `/hotzones` 页面
  - [ ] 显示所有热区列表
  - [ ] 能按状态过滤（Pending/Reviewed/Archived）
  - [ ] 能搜索热区
  - [ ] 能分页显示（每页 20 个）

#### 4. 转录功能
- [ ] **转录显示**
  - [ ] 热区包含转录文本
  - [ ] 转录文本显示在右侧转录面板
  - [ ] 点击转录词能跳转到对应时间点

- [ ] **转录缓存**
  - [ ] 同一音频片段的转录被缓存
  - [ ] 第二次加载同一热区转录立即显示

#### 5. 性能指标
- [ ] **加载时间**
  - [ ] 首页加载 < 2 秒
  - [ ] 搜索结果返回 < 2 秒（首次）
  - [ ] 搜索结果返回 < 0.5 秒（缓存）
  - [ ] 播放器加载 < 3 秒

- [ ] **音频加载**
  - [ ] 音频开始加载 < 1 秒
  - [ ] 音频能在 10 秒内开始播放
  - [ ] 音频能流畅播放（无卡顿）

- [ ] **数据库查询**
  - [ ] 热区查询 < 500ms
  - [ ] 复盘页面加载 < 2 秒

## 🔍 风险测试（高优先级）

### 1. 用户隔离安全性
- [ ] **数据隔离**
  - [ ] 用户 A 无法看到用户 B 的热区
  - [ ] 用户 A 无法修改用户 B 的热区
  - [ ] 用户 A 无法删除用户 B 的热区

- [ ] **认证检查**
  - [ ] 未登录用户无法创建热区
  - [ ] 未登录用户无法访问 `/profile`
  - [ ] 登出后无法访问受保护的页面

### 2. 数据一致性
- [ ] **跨设备同步**
  - [ ] 在设备 A 创建的热区在设备 B 可见
  - [ ] 在设备 A 修改的热区在设备 B 立即更新

- [ ] **状态管理**
  - [ ] 切换剧集后播放器状态正确重置
  - [ ] 进度条位置正确显示
  - [ ] 播放按钮状态正确显示

### 3. 边界情况
- [ ] **长音频**
  - [ ] 播放 > 1 小时的音频能正常工作
  - [ ] 进度条拖拽到末尾能正常工作

- [ ] **多热区**
  - [ ] 同一剧集有 > 10 个热区能正常显示
  - [ ] 热区列表滚动流畅（60fps）

- [ ] **网络中断**
  - [ ] 网络中断时播放器能继续播放（缓冲的部分）
  - [ ] 网络恢复后能继续加载
